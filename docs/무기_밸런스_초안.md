# 무기 밸런스 초안 (v0.1)

## 1) 기본 원리
- 스테이지가 오를수록 데미지/체력 요구량이 높아지므로 무기 수치도 단계적으로 스케일링
- 무기 레벨 1~6 적용, 누적 배율은 **최대 약 +150%**를 기준으로 수렴
- 증강은 즉시 적용되며 제거 불가
- 동일 속성/유형 중복은 상한치 + 감소계수로 완화

## 3) 레벨 성장 곡선(초안)
- 레벨 증가별 배율:
  - 레벨1: x1.00
  - 레벨2: x1.25
  - 레벨3: x1.45
  - 레벨4: x1.65
  - 레벨5: x1.85
  - 레벨6: x2.00
- 최대값 보정: 특정 무기 특성은 `+150%` 규칙에 맞춰 **최종 체감 1.5배**로 유지되도록 `maxLevel`에서 하드캡 적용

## 4) 증강 가중치(초안)

### 공격력 증강
- +10%/단계, 최대 4스택
- 한 무기 동시 적용 최대치 +25%

### 발사속도 증강
- 쿨다운 -8%/스택
- 3스택 이후 체감감소 60%

### 사거리/범위 증강
- 5%~7%/스택, 최대 3스택

### 이동속도 증강
- 플레이어 +4%/스택, 최대 4스택
- 이동계열 무기 비중 높은 구간에서만 드물게 노출

### 무기 추가/교체 증강
- 보통 런당 1회만, 슬롯이 비어 있을 때만 채택 가능

## 5) 속성 데미지 상수(초안)
- 화상 DOT: 초당 4.0, 3초 지속(기본)
- 독 DOT: 초당 2.0, 5초 지속, 단독 치명 불가(최소 1HP 보장)
- 번개 스턴: 0.4~0.7초
- 얼음 슬로우: 20~45%(스택 1), 0.8초

## 6) 시너지(2-Combination, 초안)
- 시너지는 `레벨>=5` 무기 2개 조합에서만 발동
- 발동 우선순위: 전용 상호작용 > 범용 보너스 > 부정확 효과 억제

### 추천 시너지 예시
1. 회전무기 + 화살: 정밀 타격 보정 +6%(근접 수집 대상)
2. 회전무기 + 포탑: 둔화 대상 우선 사격 +10%
3. 화살 + 블랙홀: 명중 시 흡입 반경 10% 증가
4. 장판 + 독구름: DOT 중첩 1단계 추가
5. 레이저펫 + 기절펫: 기절 대상 지속시간 +15%
6. 부메랑 + 독구름: 귀환 경로가 장판/독구름 내부에서 15% 추가 데미지

## 7) 보스 구간 스케일링 가이드
- 스테이지 10/20/.../90 구간 보스:
  - 체력 증가: 이전 구간 대비 +40~65%
- 일반구간 10단위 종료점마다:
  - 보상 XP +10%
  - 특수 적 비율 +5%

## 8) 튜닝 시 체크포인트
- 무기 4개 모두 풀 빌드 시 DPS 과다치 확인
- 독 비치명 규칙으로 인해 체력 고정 구간 과도하게 늘지 않는지 확인
- 증강 영구성으로 초반 과투자 방지: 상점/선택지 쿨타임 보정 필요
